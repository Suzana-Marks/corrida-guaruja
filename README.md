📝 Sobre o Projeto

Este é o site oficial da VIII Corrida dos Fortes 2026, realizada no Forte dos Andradas, no Guarujá/SP. O projeto foi desenvolvido sob medida para ser uma landing page de alta conversão, unindo um design moderno, limpo e focado na experiência do atleta.

O objetivo principal deste site é servir de vitrine segura e informativa, eliminando intermediários concorrentes e direcionando os usuários de forma direta e segura para a plataforma de inscrições da Olha o Ingresso.

👩‍💻 Meu Papel no Projeto

Eu trabalho na Olha o Ingresso Esportes e liderei o desenvolvimento front-end deste site parceiro. Fui responsável por:

Criar uma interface moderna inspirada nas grandes marcas esportivas (fundo claro com detalhes em verde-limão esportivo).

Estruturar a navegação de forma intuitiva dividida em duas colunas (conteúdo principal e painel de informações rápidas).

Desenvolver componentes interativos nativos (pop-ups modais para Termos de Uso e Regulamento Oficial), garantindo uma experiência fluida sem abrir novas abas.

Organizar e otimizar os caminhos de imagens e links para garantir compatibilidade local (VS Code) e produção (GitHub Pages).

⚡ Principais Funcionalidades

Arquitetura Jamstack: Site 100% estático (HTML, CSS e JavaScript), garantindo carregamento instantâneo e proteção total contra quedas ou invasões.

Modais Interativos Natividade: Pop-ups leves de Termos de Uso e Regulamento Oficial que abrem diretamente na tela do usuário.

Layout Totalmente Responsivo: Design adaptável otimizado para celulares, tablets e computadores.

Checkout Seguro Integrado: Botões com a nomenclatura ideal do segmento ("Inscreva-se Aqui") integrados à API da plataforma de vendas.

SSL Ativo (HTTPS): Criptografia de ponta a ponta na entrega do site através da infraestrutura do GitHub Pages.

🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores e mais leves tecnologias de desenvolvimento web moderno:

HTML5: Estrutura semântica e acessível.

Tailwind CSS: Framework utilitário para estilização rápida, moderna e sem arquivos CSS pesados.

JavaScript (Vanilla): Lógica limpa e nativa para manipulação dos modais interativos, sem dependência de bibliotecas pesadas.

FontAwesome 6: Biblioteca de ícones vetoriais de alta resolução.

Google Fonts (Montserrat): Tipografia premium que reforça o tom institucional e sofisticado.

💻 Como Rodar este Projeto Localmente

Se você deseja testar o site no seu computador:

Baixe os arquivos do repositório.

Certifique-se de que o arquivo index.html e as imagens estão na mesma pasta.

Abra a pasta no VS Code.

Se tiver a extensão Live Server instalada, clique com o botão direito no arquivo index.html e selecione "Open with Live Server".

O site abrirá no seu navegador padrão no endereço http://127.0.0.1:5500/.
